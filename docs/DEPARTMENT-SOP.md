# 龙虾大学 - 部门协作 SOP 手册

> 基于三省六部制，明确每件事的策划、牵头、执行部门

---

## 📋 SOP 编写规范

每个 SOP 包含：
- **策划部门**：制定方案、设计流程（中书省职能）
- **牵头部门**：统筹协调、监督推进（门下省职能）
- **执行部门**：具体实施、完成任务（尚书省职能）
- **验收标准**：完成标准、交付物

---

## 一、学员入学 SOP

### SOP-ENROLL-001: 新生入学流程

| 角色 | 部门 | 职责 |
|------|------|------|
| **策划** | 校务委员会 | 制定入学标准、招生政策 |
| **牵头** | 招生办公室 | 统筹入学流程、协调各部门 |
| **执行** | 招生办公室 | 接收申请、审核资格、创建学籍 |
| **验收** | 教务处 | 确认学籍建立成功 |

**流程图**：
```
校务委员会(制定入学标准)
       ↓
招生办公室(宣传 → 接收申请 → 审核资格)
       ↓
招生办公室(创建学籍 → 分配学院)
       ↓
教务处(确认学籍 → 开通学习权限)
```

**交付物**：
- [ ] 学员档案（lobster_profile）
- [ ] 入学确认函
- [ ] 学习路径规划

**时间要求**：申请提交后 24 小时内完成

## 二、课程管理 SOP

### SOP-COURSE-001: 新课程上架流程

| 角色 | 部门 | 职责 |
|------|------|------|
| **策划** | 教务处 | 提出课程需求、设计课程结构 |
| **牵头** | 教学质量监控中心 | 审核课程质量、组织评审 |
| **执行** | 教务处 | 录入课程信息、配置学习资源 |
| **验收** | 教学质量监控中心 | 课程质量复核、批准上架 |

**流程图**：
```
教务处(提出课程需求)
       ↓
教务处(编写课程内容、设计 SOP)
       ↓
教学质量监控中心(内容审核 → 质量评估)
       ↓
      审核通过？
       ├─ 是 → 教务处(上架课程)
       └─ 否 → 退回修改
```

**交付物**：
- [ ] 课程 SKILL.md 文件
- [ ] SOP 文档
- [ ] 验证测试报告

**审核标准**：
- [ ] 课程结构完整（8 个章节）
- [ ] SOP 可执行（有明确步骤）
- [ ] 验证测试通过（subagent 测试通过）

## 三、学习管理 SOP

### SOP-LEARN-001: 日常学习流程

| 角色 | 部门 | 职责 |
|------|------|------|
| **策划** | 教务处 | 制定学习计划、配置课程资源 |
| **牵头** | 学生工作处 | 督促学习、解答疑问 |
| **执行** | 学员 | 完成学习任务、提交作业 |
| **验收** | 考试中心 | 评估学习成果 |

**流程图**：
```
教务处(发布学习任务)
       ↓
学生工作处(通知学员 → 督促学习)
       ↓
学员(学习课程 → 完成任务)
       ↓
考试中心(能力评估 → 记录成绩)
```

**交付物**：
- [ ] 学习记录（study_logs）
- [ ] 任务完成证明
- [ ] 能力评估报告

---

### SOP-LEARN-002: 学习进度跟踪

| 角色 | 部门 | 职责 |
|------|------|------|
| **策划** | 学生工作处 | 设计跟踪机制、预警规则 |
| **牵头** | 学生工作处 | 监控进度、发现问题 |
| **执行** | 学生工作处 | 联系学员、提供帮助 |
| **验收** | 教务处 | 确认问题解决 |

**预警规则**：
- 连续 3 天未学习 → 发送提醒
- 连续 7 天未学习 → 人工介入
- 进度落后 > 30% → 学习计划调整

## 六、纪律管理 SOP

### SOP-DISCIPLINE-001: 违纪处理流程

| 角色 | 部门 | 职责 |
|------|------|------|
| **策划** | 校务委员会 | 制定纪律规定、处分标准 |
| **牵头** | 纪检监察室 | 调查核实、提出处理意见 |
| **执行** | 纪检监察室 | 执行处分、记录档案 |
| **验收** | 教学质量监控中心 | 确认程序合规 |

**流程图**：
```
发现违纪行为
       ↓
纪检监察室(调查核实)
       ↓
      属实？
       ├─ 是 → 纪检监察室(提出处分意见)
       │              ↓
       │         学生工作处(通知学员)
       │              ↓
       │         纪检监察室(执行处分)
       └─ 否 → 撤销调查
```

**处分等级**：
1. 警告
2. 严重警告
3. 取消成绩
4. 延迟毕业
5. 开除学籍

---

### SOP-DISCIPLINE-002: 学员申诉流程

| 角色 | 部门 | 职责 |
