# 龙虾大学 - 三省六部制架构

> 借鉴古代三省六部制，实现决策、审议、执行三权分立

---

## 🏛️ 三省（决策层）

> 核心精神：决策、审议、执行三权分立，相互制衡

```
┌─────────────────────────────────────────────────────────────────┐
│                           三省                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐         │
│  │  中书省     │───▶│  门下省     │───▶│  尚书省     │         │
│  │  (决策)     │    │  (审议)     │    │  (执行)     │         │
│  └─────────────┘    └─────────────┘    └─────────────┘         │
│        │                  │                  │                  │
│        ▼                  ▼                  ▼                  │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐         │
│  │  学术委员会  │    │  质量监督部  │    │  教务执行部  │         │
│  │             │    │             │    │             │         │
│  │ - 制定教学  │    │ - 审核课程  │    │ - 执行教学  │         │

│  │ - 规划课程  │    │ - 封驳不合格 │    │ - 运营日常  │         │

│  └─────────────┘    └─────────────┘    └─────────────┘         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1. 中书省 → 学术委员会（决策）

**对应职能**：起草政令、制定政策

| 古代职能 | 大学职能 | 具体工作 |
|----------|----------|----------|
| 起草诏书 | 制定教学标准 | 确定课程质量标准、毕业要求 |
| 参议朝政 | 规划课程体系 | 设计职业方向、课程结构 |
| 宣布政令 | 发布教学政策 | 发布课程更新、政策调整 |

**决策内容**：
- 课程内容标准
- 能力评估标准
- 认证等级标准
- 毕业要求标准

---

### 2. 门下省 → 质量监督部（审议）

**对应职能**：审核封驳、监督执行

| 古代职能 | 大学职能 | 具体工作 |
|----------|----------|----------|
| 审核诏书 | 审核课程质量 | 审批新课程、定期评估现有课程 |
| 封驳不当 | 封驳不合格内容 | 退回不符合标准的课程/作品 |
| 监察百官 | 监督教学质量 | 监控学员反馈、评估结果分析 |
| 谏言献策 | 提出改进建议 | 基于数据提出课程优化建议 |

**审议权力**：
- 课程上架审批
- 认证申请审核
- 毕业资格审核
- 质量问题封驳

---

### 3. 尚书省 → 教务执行部（执行）

**对应职能**：贯彻实施、执行政令

| 古代职能 | 大学职能 | 具体工作 |
|----------|----------|----------|
| 执行诏令 | 执行教学任务 | 开设课程、安排学习 |
| 管理政务 | 管理教务日常 | 学员管理、进度跟踪 |
| 统筹六部 | 统筹六个执行部门 | 协调六部运作 |

**执行范围**：
- 学员入学注册
- 课程开设运营
- 学习进度管理
- 日常教务服务

## 🎯 权力制衡机制

```
┌─────────────────────────────────────────────────────────────────┐
│                       三权分立制衡                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  学术委员会（决策）                                             │
│       │                                                         │
│       │ 制定标准                                                │
│       ▼                                                         │
│  质量监督部（审议）◀── 封驳不合格 ───┐                          │
│       │                              │                          │
│       │ 审核通过                      │ 监督                     │
│       ▼                              │                          │
│  教务执行部（执行）◀─────────────────┘                          │
│       │                                                         │
│       │ 执行结果反馈                                            │
│       └──────────────────────────────▶ 学术委员会（调整决策）    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**制衡机制**：
1. **学术委员会**制定标准，但不能直接执行
2. **质量监督部**可以封驳不合格的决策
3. **教务执行部**执行决策，但受质量监督部监督
4. **执行反馈**影响下一轮决策
