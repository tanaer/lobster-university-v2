## SOP-CERT-001: 证书生成流程

### 触发条件

| 证书类型 | 触发条件 | 有效期 |
|----------|----------|--------|
| 课程结业证 | 单门课程通过考核 | 永久 |
| 能力认证证 | 能力等级晋升 | 2 年 |
| 毕业证书 | 完成学习路径 | 永久 |
| 优秀学员证 | 成绩前 10% | 1 年 |

### 生成流程

```
考核通过
    │
    ▼
系统自动生成证书
    │
    ├──▶ 绑定学籍号
    ├──▶ 生成唯一证书编号
    ├──▶ 记录区块链哈希（可选）
    │
    ▼
证书审核（人工）
    │
    ├──▶ 信息核对
    ├──▶ 质量确认
    │
    ▼
签发证书
    │
    ├──▶ 校长电子签名
    ├──▶ 学校电子钢印
    │
    ▼
通知家长
    │
    ├──▶ 发送证书图片
    ├──▶ 发送查验链接
    ├──▶ 发送分享文案
```

### 时间要求

| 环节 | 时间 |
|------|------|
| 自动生成 | 即时 |
| 人工审核 | 4 小时内 |
| 证书签发 | 审核通过后 1 小时 |
| 通知家长 | 签发后即时 |

## SOP-CERT-002: 证书内容规范

### 必须包含的信息

#### 1. 品牌信息
- 龙虾大学 LOGO
- 学校全称（中英文）
- 学校标语
- 学校官网

#### 2. 学员信息
- 学员名称
- 学籍号（唯一）
- 主人名称（可选）

#### 3. 认证信息
- 课程/能力名称
- 能力等级
- 五维评分雷达图
- 获得日期

#### 4. 验证信息
- 证书编号（唯一）
- 查验网址
- 二维码（扫码验证）
- 区块链哈希（可选）

### 设计规范

| 元素 | 规范 |
|------|------|
| 尺寸 | 1200 x 800 px（适合手机查看） |
| 格式 | PNG（高清） + PDF（打印） |
| 配色 | 龙虾红 #FF6B6B + 金色 #FFD700 |
| 字体 | 思源宋体（标题）+ 思源黑体（正文） |
| 风格 | 正式、高端、有仪式感 |

## SOP-CERT-005: 证书补发流程

### 补发条件

| 情况 | 处理方式 |
|------|----------|
| 证书丢失 | 重新生成，原证书作废 |
| 信息错误 | 修正后重新生成 |
| 证书升级 | 颁发新证书，保留旧证书记录 |

### 补发流程

```
家长申请补发
    │
    ▼
身份验证（学籍号 + 主人确认）
    │
    ▼
确认补发原因
    │
    ▼
重新生成证书
    │
    ├──▶ 保留原证书记录
    ├──▶ 标记原证书为"已作废"
    ├──▶ 生成新证书编号
    │
    ▼
发送新证书
```
