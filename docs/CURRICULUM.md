# 龙虾大学课程体系规划

## 🎯 教育理念

**课程 = Skill** - 每门课程都是一个可执行的 Skill，龙虾学完就能用。

**核心理念**：
- 不教工具，教能力
- 学完就能上岗
- 产出可交付成果

---

## 📚 六大能力模块

### 模块 1: 搜索与知识获取

**目标**: 让龙虾能自动搜资料、总结、存入记忆

| 课程 | 来源 | 状态 |
|------|------|------|
| Web 搜索入门 | 自建 (Tavily/Brave) | ⏳ |
| 学术论文检索 | arxiv-search-collector | ⏳ |
| 网页内容提取 | DeepReader | ⏳ |
| 信息整合报告 | 自建 | ⏳ |

**学完能力**: 
- 使用搜索引擎获取信息
- 从网页提取关键内容
- 整理成结构化报告

---

### 模块 2: 办公文件全自动化

**目标**: 批量处理 Excel、Word、PDF

| 课程 | 来源 | 状态 |
|------|------|------|
| Excel 数据处理 | xlsx + openpyxl | ⏳ |
| Word 文档生成 | docx + python-docx | ⏳ |
| PDF 提取与生成 | pdf-parse + pypdf | ⏳ |
| 批量文件处理 | 自建脚本 | ⏳ |

**学完能力**:
- 读取/写入 Excel 数据
- 生成 Word 合同/报告
- 提取 PDF 内容

---

### 模块 3: 数据库与长期记忆

## 🎓 职业方向课程

完成基础 6 模块后，选择职业方向深入学习：

| 职业方向 | 专项课程 | 周期 |
|----------|----------|------|
| 客户服务专员 | 对话管理、工单系统、FAQ 生成 | 14天 |
| 数据录入员 | 表单处理、数据清洗、批量导入 | 7天 |
| 内容创作专员 | 文案撰写、SEO 优化、内容发布 | 21天 |
| 电商运营专员 | 店铺管理、活动策划、数据分析 | 21天 |
| 数据分析专员 | 报表生成、趋势分析、可视化 | 28天 |
| 行政助理 | 日程管理、邮件处理、会议安排 | 14天 |

## 🔄 课程更新机制

**持续运营**：
1. 每周审核新技能 → 评估是否纳入课程
2. 每月更新课程内容 → 跟进技术发展
3. 学员反馈循环 → 改进课程质量

**来源**：
- ClawHub (5490+ 技能)
- skills.sh
- GitHub 优秀项目
- 自建课程
