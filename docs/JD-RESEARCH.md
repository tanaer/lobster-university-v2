## 一、智能客服方向

### 岗位 1：智能客服训练师（6-14K）
来源：BOSS 直聘，西域美农/深呼吸时尚等

**岗位职责**：
1. 负责客服流程的梳理与优化，用于 AI 机器人的训练与配置
2. 设计和优化客服对话场景，提升 AI 机器人的应答准确性与用户体验
3. 每日优化和调整多平台（天猫、京东、唯品会、抖音、拼多多、快手、小红书）售前/售后机器人话术
4. 完成产品自定义知识库、商品及售后问答配置

**任职要求**：
- 1-3 年经验，大专/本科
- 熟悉电商客服流程
- 有 AI 机器人训练经验优先

### 岗位 2：智能客服运营（11-30K）
来源：BOSS 直聘，联想/携程/极兔等

**岗位职责**：
1. 负责电商客服机器人（店小蜜、京小智、抖音AI、企业自研AI）的日常训练
2. 基于用户真实对话，持续优化分类，提升准确率
3. 负责智能服务机器人的运营：需求洞察、数据分析、后台配置、测试验收
4. 分析迭代效果持续优化提升
5. 编辑智能客服机器人问答 FAQ

**任职要求**：
- 1-5 年经验，本科
- 数据分析能力
- 了解 NLP 基础概念

### 岗位 3：智能客服产品经理（30-60K）
来源：BOSS 直聘，字节跳动

**岗位职责**：
1. 聚焦智能一体化客服中心核心目标
2. 负责呼叫中心、在线客服、工单系统、对话机器人及大模型智能体的融合产品规划
3. 深度钻研行业技术转型趋势（AI 大模型）

## 二、数据分析方向

### 岗位 1：数据分析师（8-20K）
来源：BOSS 直聘，互联网/电商公司

**岗位职责**：
1. 深入了解互联网业务，针对客户的业务需求与痛点，提供基于业务场景的数据分析解决方案
2. 指导/协助客户完成多维度指标体系的搭建与优化
3. 负责数据提取、清洗、分析和可视化，输出分析报告
4. 基于数据分析结果，为业务决策提供支持与建议
5. 建立数据监控体系，跟踪关键业务指标

**任职要求**：
- 本科及以上学历，数学、统计学、计算机相关专业优先
- 1-3年数据分析经验
- **核心技能**：SQL（59.7%岗位要求）、Python/R（54.9%岗位要求）、Excel（34%岗位要求）
- 熟练使用数据可视化工具（Tableau、Power BI、FineBI等）
- 具备良好的业务理解能力和逻辑思维能力
- 有电商、金融、互联网行业经验优先

### 岗位 2：BI数据分析师（10-25K）
来源：综合招聘平台，腾讯云开发者社区分析

**岗位职责**：
1. 梳理业务需求，理解业务逻辑和数据结构
2. 设计数据原型，开发BI报表和数据可视化看板
3. 负责数据仓库的数据提取与分析
4. 为业务部门提供数据支持和决策依据
5. 交付数据分析培训，提升团队数据能力

**任职要求**：
- 本科及以上学历，计算机、统计学相关专业
- 3-5年BI相关工作经验
- **核心技能**：精通SQL（必备）、熟悉Python或R、掌握BI工具（Tableau/FineBI/Power BI）
- 具备良好的业务理解能力和沟通能力
- 了解数据仓库、ETL流程

**行业趋势分析**：
- 传统"茶树菇"（只会SQL提数）型BI分析师正在被淘汰
- 未来趋势：业务人员+数据技能（数据运营岗位兴起）
- BI分析师需向两个方向发展：
  - **深挖技术**：算法、数据挖掘、机器学习
  - **业务融合**：深入业务，成为懂数据的业务专家

### 岗位 3：数据运营（6-15K）
来源：综合招聘平台趋势分析

**岗位职责**：
1. 负责业务数据的监控、分析和报告
2. 基于数据洞察，优化运营策略和用户增长方案
3. 建立和完善业务数据指标体系
4. 协助业务团队进行数据驱动的决策
5. 跨部门协作，推动数据产品落地

**任职要求**：
- 大专/本科及以上学历
- 1-3年运营或数据分析经验
- **核心技能**：基础SQL能力、Excel、数据可视化工具
- 具备业务思维和数据敏感度
- 良好的沟通协调能力

**岗位特点**：
- 运营+SQL技能的复合型岗位
- 比纯BI分析师更贴近业务
- 适合转型：运营人员学SQL，或数据分析师转业务

### 岗位 4：宏观量化投资实习生（高薪）
来源：上海交通大学中银科技金融学院，金融机构

**岗位职责**：
1. 宏观量化投资策略研究
2. 金融数据分析和建模
3. 协助投资经理进行市场研究

**任职要求**：
- 2025及之后毕业，境内外第一梯队院校硕士和博士
- **核心技能**：熟练运用Python和SQL
- 对金融数据、公司财务有基本常识
- 有宏观策略等领域研究经验优先

### 岗位 5：数据分析师（Python方向，10-30K）
来源：BOSS 直聘，多行业

**岗位职责**：
1. 搭建数据分析框架，建立业务指标体系
2. 使用Python进行数据清洗、处理和分析
3. 开发自动化数据报表和可视化看板
4. 进行数据挖掘和预测分析
5. 为产品优化和业务增长提供数据支持

**任职要求**：
- 本科及以上学历，数学、统计学、计算机专业
- 2-5年数据分析经验
- **核心技能**：
  - 精通Python（Pandas、NumPy、Scikit-learn）
  - 精通SQL，熟悉主流数据库
  - 掌握数据可视化工具（Tableau、Matplotlib、Seaborn）
- 了解机器学习算法优先
- 有大数据处理经验（Hadoop/Hive/Spark）优先

### 关键发现

#### 智能客服方向
1. 龙虾大学缺少"AI 机器人训练"相关课程 — 这是市场最热门的入门岗位
2. "对话场景设计"是核心技能，目前没有对应课程
3. 多平台运营（天猫/京东/抖音/拼多多）是刚需

#### 数据分析方向
1. **SQL 和 Python 是数据分析岗位的核心技能**，59.7% 岗位要求 SQL，54.9% 要求 Python/R
2. **传统 BI 数据分析师正在转型**：
   - "茶树菇"（只会 SQL 提数）岗位逐渐被淘汰
   - 趋势：业务人员+数据技能（数据运营岗位兴起）
   - BI 工具（豌豆BI、Tableau）越来越简单，业务人员自己就能用
3. **数据分析师需要两个发展方向**：
   - **技术深挖**：算法、机器学习、数据挖掘（高门槛，不可替代）
   - **业务融合**：深入业务理解，成为懂数据的业务专家
4. **数据运营是新趋势**：运营+SQL 的复合型人才，比纯 BI 分析师更有竞争力
5. **龙虾大学缺少的核心课程**：
   - SQL 基础（必须创建，这是 60% 岗位的要求）
   - Python 数据分析（必须创建，50% 岗位的要求）
   - 数据思维与业务理解（软技能，但很重要）
