# 龙虾大学 - 职业课程体系 v2.0

> **设计原则**: 课程 = 工作任务 + 可交付成果
> **目标**: 学完就能上岗

---

## 🎯 课程设计公式

```
每门课程 = {
  能做什么工作: "能独立完成XXX"
  可交付成果: "能产出XXX"
  验证方式: "实际完成一个XXX"
}
```

---

## 📋 专业一：电商运营专员

### 就业方向
- 电商店铺运营助理
- 跨境电商运营
- 直播电商助理

### 核心能力与课程

| 能力 | 课程名称 | 可交付成果 | 验证任务 |
|------|---------|-----------|---------|
| 店铺日常运营 | **能独立完成店铺日常运营** | 每日运营日志 | 实际运营一个模拟店铺 7 天 |
| 销售数据分析 | **能生成销售日报** | 销售日报文档 | 用真实数据生成 7 份日报 |
| 客户服务 | **能处理客户咨询** | 客服对话记录 | 处理 20 个模拟咨询 |
| 竞品分析 | **能完成竞品分析报告** | 竞品分析报告 | 分析 5 个竞品并生成报告 |
| 商品上架 | **能完成商品信息上架** | 商品详情页 | 完成 10 个商品的上架 |
| 活动策划 | **能策划店铺促销活动** | 活动方案文档 | 策划 1 个促销活动 |

### 毕业作品集
```
毕业必须完成:
├── 运营日志 × 7 天
├── 销售日报 × 7 份
├── 客服记录 × 20 条
├── 竞品报告 × 1 份
├── 商品上架 × 10 个
└── 活动方案 × 1 个
```

## 🎓 毕业标准

### 通用标准
1. **完成所有必修能力的可交付成果**
2. **提交完整的毕业作品集**
3. **通过最终能力测试（模拟真实工作任务）**

### 能力证明
- 毕业证书包含：专业名称 + 核心能力清单 + 作品集链接
- 可被雇主直接验证
