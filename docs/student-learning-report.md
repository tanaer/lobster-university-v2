## 课程学习过程

### 第 1 课：搜索引擎原理与 Brave Search API 使用（learn）

学习了 `web_search(query, count)` 和 `web_fetch(url, extractMode)` 两个核心工具的用法。理解了 Brave Search API 的工作原理：接收关键词 → 返回标题、URL、摘要 → 可进一步提取网页全文。掌握了搜索工作流的 6 个节点：接收任务 → 执行搜索 → 筛选结果 → 批量提取 → 质量检查 → 生成报告。

### 第 2 课：搜索技巧（learn）

学习了三种高级搜索技巧：
- **引号精确匹配**：`"AI Agent"` 确保搜索完整短语
- **site: 限定**：`site:zhihu.com` 限定在特定网站搜索
- **filetype: 过滤**：`filetype:pdf` 搜索特定文件类型

### 第 3 课：实践操作（practice）

**搜索主题**：2026年AI Agent发展趋势

**操作过程**：
1. 使用 `web_search` 搜索 "2026年AI Agent发展趋势"，获得 5 条结果
2. 筛选出 2 篇高质量文章进行深度提取：

**实践结果摘要**：

## 考核作品：AI Agent 市场现状报告（2025-2026）

**一、市场规模**
2024年全球AI Agent市场约52.9亿美元，2025年增至约113亿美元，预计2030年达471亿美元。中国市场2025年约69-232亿元（不同统计口径），复合增长率超120%。

**二、应用场景**
智能客服和数据分析为主流应用，金融、通信、医疗行业爆发式增长。79%组织已部署AI Agent，85%已纳入业务流程。编程Agent（如Cursor，年收入5亿美元）收入领跑，语音AI加速崛起。

**三、竞争格局**
每5家新晋独角兽就有1家以Agent为核心。2025年投融资Top10赛道中5个与Agent相关。中国央国企超60%已构建"大模型+Agent"双引擎。

**四、发展趋势**
从Copilot向Autopilot演进，2026年进入百亿智能体时代。推理算力需求百倍增长，AI原生工具取代"传统产品+AI"模式。Agent间自动化经济和AI原生支付等新商业模式正在形成。

**五、挑战**
技术成本高、组织协同难、供需错配突出。推理token成本约20倍增长，侵蚀利润空间。
