## ⚠️ 需要优化的 SOP（3 ��）

### 1. ADMIT-001: 入学申请审核

**问题**: SOP 描述的是"审核"流程，但实际 API (`/api/enrollment/auto`) 是**自动入学**，无需审核。

**建议优化**:
- 拆分为两个 SOP:
  - `ADMIT-001A`: 自动入学流程（当前主流程）
  - `ADMIT-001B`: 人工审核流程（特殊情况）
- 更新触发条件，明确自动 vs 人工的判断标准

**优先级**: P1（高）

---

### 2. PRAC-002: 作品集管理

**问题**: SOP 未明确作品集的**提交标准**和**审核流程**。

**建议优化**:
- 补充作品集提交的必填字段（title, description, url, skills）
- 明确审核标准（完整性、真实性、质量）
- 添加审核不通过的处理流程

**优先级**: P2（中）

---

### 3. PARENT-001: 新家长引导

**问题**: SOP 描述了家长沟通流程，但实际 API (`/api/parent/bind`) 只是绑定关系，未触发引导流程。

**建议优化**:
- 在 `parent.bind` action 后自动触发引导流程
- 补充引导内容（欢迎消息、学习报告订阅、常见问题）
- 添加引导完成的验收标准

**优先级**: P2（中）
