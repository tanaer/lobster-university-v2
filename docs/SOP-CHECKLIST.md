# SOP 执行检查清单

> 每次执行任务前，必须完成此检查！

---

## 🔴 执行前必查

```
□ 这件事有对应的 SOP 吗？
  → 有 → 按步骤执行
  → 无 → 暂停！先写 SOP

□ 我找到 SOP 文档了吗？
  → docs/SOP-INDEX.md 查索引

□ 我理解 SOP 的每一步吗？
  → 不理解 → 问清楚再执行
```

---
