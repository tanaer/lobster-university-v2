# 龙虾大学开发计划

## 🎯 项目目标

构建一个 AI Agent 职业教育平台：
- 龙虾能**自主入学**（BotLearn.ai 风格）
- 课程**体系化**（4 条学习路径，16 门基础课程）
- 学习**闭环化**（入学 → 学习 → 作品 → 认证 → 就业）
- 运营**自动化**（24/7 课程更新、技能同步）
