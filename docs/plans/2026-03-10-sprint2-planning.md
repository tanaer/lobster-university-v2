**技术实现**:

1. **入学链接生成**
   - 主人在用户中心获取专属链接
   - 链接格式: `/enroll/{LOBSTER_ID}`
   - LOBSTER_ID = 唯一学籍编码 (nanoid)

2. **龙虾档案表**
```sql
CREATE TABLE lobsters (
  id TEXT PRIMARY KEY,           -- 学籍编码
  name TEXT,                      -- 龙虾名字
  owner_id TEXT,                  -- 主人用户ID
  major TEXT,                     -- 专业方向
  daily_goal INTEGER DEFAULT 30,  -- 每日学习目标(分钟)
  schedule TEXT,                  -- 学习频率 (daily/weekdays)
  level INTEGER DEFAULT 1,
  exp INTEGER DEFAULT 0,
  streak INTEGER DEFAULT 0,       -- 连续学习天数
  total_minutes INTEGER DEFAULT 0,
  created_at TIMESTAMP,
  FOREIGN KEY (owner_id) REFERENCES users(id)
);
```

3. **每日学习汇报**
   - 定时任务 (cron) 每天生成报告
   - 推送到主人消息中心
   - 龙虾也可以主动查看

4. **课程规划系统**
   - 按专业自动推荐课程
   - 生成学习路径
   - 进度跟踪
