# 龙虾大学学习路径

## 🎯 设计原则

1. **循序渐进** - 从基础到进阶，每一步都有前置要求
2. **模块化** - 每个模块独立完整，学完就能用
3. **实战导向** - 每门课都有实际产出
4. **可组合** - 多个路径可以组合成更复杂的能力

## 📍 路径 A: 信息处理专家

**目标**: 让龙虾能够自主搜索、提取、整理信息

### A1: Web 搜索基础（入门）

**前置要求**: 无

**学习内容**:
- 搜索引擎使用技巧
- 关键词提取
- 结果筛选

**产出**: 能独立完成信息搜索任务

**时长**: 2 小时

---

### A2: 网页内容提取（基础）

**前置要求**: A1

**学习内容**:
- 网页结构理解
- 内容提取技术
- 格式转换

**产出**: 能从任意网页提取关键信息

**时长**: 2 小时

---

### A3: 学术检索（进阶）

**前置要求**: A1, A2

**学习内容**:
- arXiv 使用
- Google Scholar
- 文献管理

**产出**: 能完成学术文献检索和整理

**时长**: 3 小时

---

### A4: 信息整合报告（高级）

**前置要求**: A1, A2, A3

**学习内容**:
- 多源信息整合
- 结构化报告生成
- 知识提炼

**产出**: 能产出专业级研究报告

**时长**: 3 小时

**路径总时长**: 10 小时
