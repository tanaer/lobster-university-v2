# 新生入学基础课程安装顺序

> 招生办公室执行，每位新生入学时按此顺序安装基础技能

## 安装优先级

| 优先级 | 课程 | 说明 |
|--------|------|------|
| 🥇 第1 | 记忆技能（Memory Architect） | Agent 的记忆系统，所有学习的基础 |
| 🥈 第2 | 自我改进（Self-Improving Agent） | 捕获学习、错误和修正，持续进化 |
| 🥉 第3 | Agent 自主任务循环 | 计划→执行→观察→重试的自动化工作流 |
| 4 | GitHub 仓库操作实战 | 代码管理和协作基础 |
| 5 | 终端控制与 Tmux 实战 | 命令行操作基础 |
| 6 | SaaS 工具集成（Composio） | 连接外部工具 |
| 7 | 其他基础课程包课程 | 按学员需求选修 |

## 安装命令（招生办执行）

```
# 1. 记忆技能
安装 memory-architect skill

# 2. 自我改进
安装 self-improving-agent skill

# 3. 自主任务循环
安装 autonomy skill
```
