## SOP-QA-004: 课程效果评估流程

| 项目 | 内容 |
|------|------|
| **触发** | 课程完成、季度评估 |
| **策划** | 教学质量监控中心 |
| **牵头** | 教学质量监控中心 |
| **执行** | 考试中心 |
| **验收** | 校务委员会 |

### 评估维度
1. **完成率** - 学员完成课程的比例
2. **通过率** - 通过考核的学员比例
3. **满意度** - 学员评分（1-5 星）
4. **就业率** - 毕业学员就业比例
5. **能力提升** - 学习前后能力对比

### 流程步骤
1. 收集课程数据（完成率、通过率）
2. 发送满意度问卷
3. 分析考核成绩
4. 追踪毕业学员去向
5. 生成评估报告
6. 提出改进建议
7. 反馈给教务处

### 评估标准
| 指标 | 优秀 | 合格 | 需改进 |
|------|------|------|--------|
| 完成率 | >80% | >60% | <60% |
| 通过率 | >85% | >70% | <70% |
| 满意度 | >4.5 | >3.5 | <3.5 |
| 能力提升 | >30% | >15% | <15% |

### 输入
- 课程学习数据
- 考核成绩
- 学员反馈

### 输出
- 课程评估报告
- 改进建议
- 优秀课程表彰

### 时间要求
- 数据收集: 3 天
- 分析报告: 2 天
- 反馈: 1 天

## SOP-QA-005: 学员满意度调查流程

| 项目 | 内容 |
|------|------|
| **触发** | 每月定期、课程完成 |
| **策划** | 学生工作处 |
| **牵头** | 学生工作处 |
| **执行** | 学生工作处 |
| **验收** | 教学质量监控中心 |

### 调查内容
1. 课程内容质量
2. 教学方式效果
3. 学习资源充足度
4. 服务响应速度
5. 整体满意度
6. 改进建议

### 流程步骤
1. 设计调查问卷
2. 发送给目标学员
3. 收集回复（7 天）
4. 统计分析结果
5. 生成调查报告
6. 提出改进措施
7. 跟踪改进效果

### 问卷模板
```
【龙虾大学满意度调查】

1. 您对课程内容质量的评分（1-5分）：
2. 教学方式是否适合您（是/否/部分）：
3. 学习资源是否充足（是/否）：
4. 问题响应是否及时（是/否）：
5. 整体满意度（1-5分）：
6. 您最满意的方面：
7. 您认为需要改进的方面：
8. 其他建议：
```

### 输入
- 学员名单
- 调查问卷

### 输出
- 满意度报告
- 改进措施清单
- 问题跟踪表

### 时间要求
- 调查周期: 7 天
- 报告: 收集后 3 天
- 改进: 根据优先级

## SOP-QA-006: SOP 审查优化流程

| 项目 | 内容 |
|------|------|
| **触发** | 每季度、发现问题 |
| **策划** | 教学质量监控中心 |
| **牵头** | 教学质量监控中心 |
| **执行** | 各部门 |
| **验收** | 校务委员会 |

### 审查内容
1. SOP 执行情况
2. 执行效果评估
3. 问题收集
4. 改进建议
5. 版本更新

### 流程步骤
1. 发起 SOP 审查通知
2. 各部门自查执行情况
3. 收集问题和建议
4. 分析问题根因
5. 制定优化方案
6. 评审通过
7. 发布新版本
8. 培训相关人员

### 审查标准
- [ ] 是否有执行记录
- [ ] 是否达到预期效果
- [ ] 是否有优化空间
- [ ] 是否与其他 SOP 冲突
- [ ] 是否需要更新

### 输入
- SOP 执行记录
- 问题反馈
- 改进建议

### 输出
- SOP 审查报告
- 优化后的 SOP
- 版本更新日志

### 时间要求
- 自查: 3 天
- 分析: 2 天
- 优化: 5 天
- 发布: 1 天
