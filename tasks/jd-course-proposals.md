### 搜集摘要

**X/Twitter 趋势（8条热门帖）：**
1. **AI Agent Engineer 岗位兴起** — Arman Hezarkhani 发布"2026 AI Agent Engineer 完整指南"，强调用 Claude Code/Cursor 构建 Agent 比手写代码快 10x
2. **Agent 产品化是最大机会** — Greg Isenberg 指出：找到重复性工作流 → 用 Agent 替代 → 最薄 UI 包装 → 按节省工时定价。核心技能：工作流提取、Prompt 工程、产品化
3. **Agent 记忆系统突破** — 微软发布 memory-augmented LLM agent 论文，混合 on-policy/off-policy 优化
4. **长任务 Agent 训练** — 字节跳动用 RL 训练 Agent 完成 27 种长链任务，媲美顶级闭源模型
5. **主动式个性化 Agent** — CMU 发布 PPP 框架（Productive, Proactive, Personalized），Agent 不只完成任务，还要主动提问、适应用户风格
6. **动态多 Agent 编排** — 静态工作流（固定角色、线性流程）在复杂场景下崩溃，需要动态通信和自适应编排
7. **Agent 架构三件套** — LLM（推理）+ RAG（记忆）+ Tools（执行）= 不是聊天机器人，是操作员
8. **Agent 物理定律** — 研究发现 LLM Agent 行为遵循宏观物理定律，可作为物理系统研究

**国内 AI Agent 生态信号：**
- 小红书自动化发布工具（MCP 集成）热度持续上升
- Agentic AI Workflow 实战教程需求旺盛（LangChain + LangGraph）
- AI Agent vs AI Workflow 概念辨析成为热门科普内容
- 非技术用户对"输入主题→全自动发布"的端到端 Agent 需求强烈
