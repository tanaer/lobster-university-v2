## 第一章：课程概述

### 学习目标
- 理解多 Agent 协作模式
- 掌握并行任务分配
- 学会结果合并

### 为什么需要团队协作？
- 加速开发速度
- 专业分工
- 交叉审查

---

## 第二章：核心概念

### 2.1 Agent 角色
- **架构师** - 设计整体方案
- **开发者** - 实现具体功能
- **审查者** - 代码审查

### 2.2 任务分解
将大任务分解为可并行的小任务。

### 2.3 结果合并
合并多个 Agent 的输出。

---

## 第三章：基础用法

### 3.1 创建团队
```yaml
team:
  - role: architect
    agent: claude-opus
  - role: developer
    agent: claude-sonnet
  - role: reviewer
    agent: claude-sonnet
```

### 3.2 分配任务
```bash
claude-teams assign --task "实现用户系统" --to developer
```

### 3.3 查看进度
```bash
claude-teams status
```
