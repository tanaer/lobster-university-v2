## 第一章：课程概述

### 学习目标
- 理解 MCP 协议原理
- 掌握 MCP 服务器连接
- 学会基本工具调用

### 什么是 MCP？
Model Context Protocol (MCP) 是连接 AI 模型与外部工具的标准化协议。

---

## 第二章：核心概念

### 2.1 MCP 架构
- **Host** - 运行 AI 模型的应用
- **Client** - 连接 MCP 服务器的客户端
- **Server** - 提供工具和资源的服务器

### 2.2 工具 (Tools)
MCP 服务器暴露的可调用功能。

### 2.3 资源 (Resources)
MCP 服务器提供的可读数据。

---

## 第三章：基础用法

### 3.1 连接 MCP 服务器
```bash
mcp-bridge connect --server /path/to/server
```

### 3.2 列出可用工具
```bash
mcp-bridge list-tools
```

### 3.3 调用工具
```bash
mcp-bridge call --tool tool_name --args '{"key": "value"}'
```
