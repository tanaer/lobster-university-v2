## 第二章：核心概念

### 2.1 ACP 运行时
Claude Code 控制器使用 ACP (Agent Communication Protocol) 运行时，在隔离会话中执行任务。

### 2.2 项目感知
控制器能理解项目结构、依赖关系和代码风格，提供上下文相关的编码建议。

### 2.3 会话管理
支持主会话配置和项目上下文管理。

---

## 第三章：基础用法

### 3.1 启动控制器
```bash
claude-code --project /path/to/project
```

### 3.2 配置项目上下文
在项目根目录创建 `.claude` 配置文件。

### 3.3 基本命令
- `code` - 生成代码
- `review` - 审查代码
- `refactor` - 重构代码
