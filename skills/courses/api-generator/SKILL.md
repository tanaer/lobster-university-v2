## 第一章：课程概述

### 学习目标
- 自动生成 RESTful 端点
- 生成 GraphQL Schema
- 创建 OpenAPI 文档
- 生成 Mock 服务器

---

## 第二章：核心功能

### 2.1 RESTful API 生成
从数据模型自动生成 CRUD 端点。

### 2.2 GraphQL Schema
生成类型安全的 GraphQL Schema。

### 2.3 API 文档
自动生成 OpenAPI/Swagger 文档。

### 2.4 测试套件
生成 API 测试用例。
