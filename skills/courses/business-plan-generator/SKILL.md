## 第二章：核心概念

### 2.1 商业计划书的完整结构

一份标准 BP 包含以下部分：

```
1. 执行摘要（Executive Summary）
   └── 一页纸概括全部内容，最后写但放在最前

2. 公司概述（Company Description）
   └── 使命、愿景、法律结构、团队

3. 市场分析（Market Analysis）
   └── TAM/SAM/SOM、目标客户、市场趋势

4. 竞争分析（Competitive Analysis）
   └── 竞品对比、差异化优势、护城河

5. 产品/服务（Products & Services）
   └── 核心功能、技术架构、发展路线图

6. 营销策略（Marketing Strategy）
   └── 获客渠道、定价策略、品牌定位

7. 运营计划（Operations Plan）
   └── 团队、流程、里程碑

8. 财务预测（Financial Projections）
   └── 收入预测、成本结构、盈亏平衡

9. 附录（Appendix）
   └── 补充数据、专利、合同等
```

### 2.2 精益画布（Lean Canvas）— Ash Maurya

精益画布是一页纸的商业模型工具，适合早期创业验证：

| # | 要素 | 核心问题 | 填写技巧 |
|---|------|----------|----------|
| 1 | Problem（问题） | 客户最痛的 Top 3 问题是什么？ | 从客户访谈中提炼，不是你猜的 |
| 2 | Customer Segments（客户群体） | 谁有这个问题？谁会付费？ | 区分目标用户和早期采用者 |
| 3 | Unique Value Proposition（独特价值主张） | 一句话说清楚你为什么不同 | 公式：为[谁]解决[什么]，不同于[竞品] |
| 4 | Solution（解决方案） | 你怎么解决每个问题？ | 对应 Problem 逐一回答 |
| 5 | Channels（渠道） | 如何触达客户？ | 免费渠道优先，付费渠道后验证 |
| 6 | Revenue Streams（收入来源） | 怎么赚钱？ | 定价模型：订阅/按量/一次性 |
| 7 | Cost Structure（成本结构） | 花多少钱？ | 固定成本 + 可变成本 |
| 8 | Key Metrics（关键指标） | 怎么衡量成功？ | 找到你的北极星指标 |
| 9 | Unfair Advantage（不公平优势） | 别人抄不走的是什么？ | 网络效应/数据/专利/团队/品牌 |

```

│   Problem    │  Solution    │   Unique     │  Unfair      │  Customer    │

│  Top 3 问题  │  对应方案     │  Proposition │              │              │
│              │              │              │  抄不走的     │  目标用户     │
│  1.          │  1.          │  一句话：     │  优势         │  早期采用者   │
│  2.          │  2.          │              │              │              │
│  3.          │  3.          │              │              │              │
├──────────────┼──────────────┤              ├──────────────┼──────────────┤
│  Key Metrics │  Channels    │              │  Cost        │  Revenue     │
│              │              │              │  Structure   │  Streams     │
│  北极星指标   │  获客渠道     │              │              │              │
│              │              │              │  固定成本     │  定价模型     │
│              │              │              │  可变成本     │              │

```

### 2.4 商业模型画布（BMC）— Alexander Osterwalder

BMC 比精益画布更适合成熟企业，9 个模块：

| 模块 | 说明 |
|------|------|
| 价值主张 | 你为客户创造什么价值？ |
| 客户细分 | 你服务哪些客户群体？ |
| 渠道 | 如何触达和交付？ |
| 客户关系 | 如何获取、维护、增长客户？ |
| 收入来源 | 客户愿意为什么付费？ |
| 核心资源 | 需要什么关键资源？ |
| 关键活动 | 必须做好哪些事？ |
| 重要伙伴 | 需要哪些合作伙伴？ |
| 成本结构 | 最大的成本是什么？ |

### 4.2 生成 SWOT 分析

**SWOT 分析框架**：

```
         正面因素              负面因素
        ┌──────────────┬──────────────┐
 内部   │  Strengths   │  Weaknesses  │
 因素   │  优势         │  劣势         │
        ├──────────────┼──────────────┤
 外部   │ Opportunities│   Threats    │
 因素   │  机会         │  威胁         │
        └──────────────┴──────────────┘
```

**示例：AI 客服 SaaS 的 SWOT**

| 维度 | 内容 |
|------|------|
| **Strengths** | ① 技术团队有 NLP 背景 ② 产品已有 MVP ③ 成本比人工低 70% |
| **Weaknesses** | ① 品牌知名度低 ② 资金有限 ③ 缺乏销售团队 |
| **Opportunities** | ① 电商市场持续增长 ② AI 接受度提高 ③ 竞品定价偏高 |
| **Threats** | ① 大厂可能入场 ② 政策监管不确定 ③ 客户对 AI 客服信任度低 |

## 第五章：进阶技巧

### 5.1 财务预测

**三年收入预测模板**：

| 指标 | 第 1 年 | 第 2 年 | 第 3 年 |
|------|---------|---------|---------|
| 付费客户数 | 100 | 500 | 2,000 |
| ARPU（月均） | ¥199 | ¥249 | ¥299 |
| 月收入 | ¥19,900 | ¥124,500 | ¥598,000 |
| 年收入 | ¥238,800 | ¥1,494,000 | ¥7,176,000 |
| 增长率 | - | 526% | 380% |

**成本结构模板**：

| 成本项 | 类型 | 第 1 年 | 第 2 年 | 第 3 年 |
|--------|------|---------|---------|---------|
| 服务器/云 | 可变 | ¥60,000 | ¥200,000 | ¥600,000 |
| AI API 调用 | 可变 | ¥120,000 | ¥400,000 | ¥1,200,000 |
| 人员工资 | 固定 | ¥480,000 | ¥960,000 | ¥1,920,000 |
| 营销 | 可变 | ¥50,000 | ¥200,000 | ¥500,000 |
| 办公/其他 | 固定 | ¥60,000 | ¥120,000 | ¥240,000 |
| **总成本** | | **¥770,000** | **¥1,880,000** | **¥4,460,000** |

**盈亏平衡分析**：

```
盈亏平衡点 = 固定成本 / (单价 - 单位可变成本)

例：
固定成本 = ¥540,000/年（人员+办公）
单价 = ¥199/月 = ¥2,388/年
单位可变成本 = ¥80/月 = ¥960/年（服务器+API）

盈亏平衡 = 540,000 / (2,388 - 960) = 378 个客户
```

### 5.3 电梯演讲（30 秒版）

**公式**：

> "我们为 **[目标客户]** 解决 **[核心问题]**，通过 **[解决方案]**。不同于 **[竞品]**，我们的优势是 **[差异化]**。目前已有 **[牵引力数据]**。"

**示例**：

> "我们为中小电商卖家解决客服成本高、响应慢的问题，通过 AI 自动客服平台。不同于通用客服工具，我们专注电商场景，AI 解决率达 85%。目前已有 50 家付费客户，月增长 30%。"

### 5.4 TAM/SAM/SOM 计算

```
TAM（总可触达市场）
= 全球/全国电商企业数 × 平均客服支出
= 500 万家 × ¥5,000/年 = ¥250 亿

SAM（可服务市场）
= 目标区域 × 目标规模的企业 × 你的价格
= 50 万家中小电商 × ¥2,388/年 = ¥12 亿

SOM（可获取市场）
= 3 年内可触达的客户 × 你的价格
= 2,000 家 × ¥2,388/年 = ¥478 万
```

## 第六章：最佳实践

### ✅ 推荐做法

1. **数据驱动**：每个数字都要有来源，不要拍脑袋
2. **客户验证**：先和 10 个潜在客户聊，再写 BP
3. **迭代更新**：BP 不是一次性文档，每月更新
4. **简洁有力**：投资人平均看 BP 不超过 3 分钟
5. **视觉化**：用图表代替大段文字
6. **保守预测**：财务预测宁可保守，不要画大饼

### ❌ 避免做法

1. **空洞描述**：避免"市场巨大"这种没有数据的说法
2. **忽略竞品**：说"没有竞争对手"是最大的红旗
3. **过度乐观**：第一年就预测百万收入不现实
4. **忽略风险**：不提风险 = 没想清楚
5. **模板化**：直接套模板不改 = 没有思考
6. **太长**：超过 20 页没人看

## 第七章：常见问题

### Q1: 精益画布和商业模型画布有什么区别？
精益画布（Lean Canvas）适合早期创业验证，聚焦问题和解决方案。商业模型画布（BMC）适合成熟企业，聚焦资源和合作伙伴。初创用 Lean Canvas，成长期转 BMC。

### Q2: 财务预测应该做几年？
标准是 3 年。第 1 年按月预测，第 2-3 年按季度或年预测。投资人知道预测不准，但他们看的是你的思考逻辑和假设是否合理。

### Q3: SWOT 分析容易犯什么错？
最常见的错误：(1) 把外部因素写到内部（如"市场大"不是 Strength），(2) 太笼统（"技术强"不如"NLP 准确率 95%"），(3) 只列不分析（列完要制定应对策略）。

### Q4: 电梯演讲多长合适？
30 秒版（核心版）和 2 分钟版（详细版）各准备一个。30 秒版用于社交场合，2 分钟版用于正式路演开场。

### Q5: AI 生成的 BP 能直接用吗？
不能。AI 生成的是框架和初稿，你需要：(1) 填入真实数据，(2) 验证市场假设，(3) 加入团队独特洞察，(4) 反复打磨语言。AI 是加速器，不是替代品。

### Q6: 投资人最看重 BP 的哪部分？
排序：(1) 团队，(2) 市场规模，(3) 牵引力数据（已有客户/收入），(4) 商业模式，(5) 竞争优势。财务预测反而排最后，因为大家都知道会变。
