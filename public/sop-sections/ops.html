SOP-OPS-002

数据备份流程

项目内容
触发每周自动（周一 03:00）
执行系统（自动化）
验收教学质量监控中心

备份范围：lobster.db · skills/ · docs/ · drizzle/

流程步骤

  1. 系统自动触发备份
  2. 创建备份目录（带时间戳）
  3. 复制所有数据文件
  4. 验证备份完整性
  5. 清理 30 天前的旧备份
  6. 发送备份报告
BACKUP_DIR="backups/$(date +%Y%m%d_%H%M%S)"
mkdir -p $BACKUP_DIR
cp lobster.db $BACKUP_DIR/
cp -r skills docs drizzle $BACKUP_DIR/
find backups -type d -mtime +30 -exec rm -rf {} \;

SOP-OPS-005

紧急事件响应流程

项目内容
触发服务宕机、数据丢失、安全事件
牵头校务委员会
执行全体

事件分级

级别定义响应时间
🔴 P0服务完全不可用15 分钟
🟠 P1核心功能不可用1 小时
🟡 P2非核心功能异常4 小时
🟢 P3体验问题24 小时

流程步骤

  1. 发现 - 初步判断级别
  2. 上报 - 通知校务委员会
  3. 响应 - 成立应急小组
  4. 处理 - 执行修复方案
  5. 恢复 - 验证服务恢复
  6. 复盘 - 编写事故报告
  7. 改进 - 更新 SOP 防止复发
