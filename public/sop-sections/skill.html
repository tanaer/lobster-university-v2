🛠️ 技能使用 SOP（7 个）

所有新安装的技能必须有对应的 SOP 才能使用。无 SOP 不使用！
SOP-SKILL-001

课程生成流程

项目内容
触发新课程需求、课程内容更新
执行工具curriculum-generator
验收教学质量监控中心

流程步骤

  1. 教务处提出课程需求（主题、目标学员、学习目标）
  2. 使用 curriculum-generator 生成课程大纲
  3. 人工审核课程结构（8 章节标准）
  4. 生成课程 SKILL.md 文件
  5. 质量监控中心审核
  6. 上架发布

质量标准

时间：生成 30 分钟 · 审核 2 小时 · 发布 24 小时内

SOP-SKILL-002

教学辅导流程

项目内容
触发学员提问、学习困难
执行工具teacher / tutor

流程步骤

  1. 接收学员问题
  2. 分析问题类型（概念/操作/实践）
  3. 调用 teacher 解释概念
  4. 调用 tutor 个性化辅导
  5. 学员确认理解
  6. 记录辅导日志

时间：响应 5 分钟 · 完成 30 分钟

SOP-SKILL-003

Agent 培训流程

项目内容
触发新 Agent 入学、技能升级
执行工具agent-training
验收考试中心

流程步骤

  1. Agent 入学注册
  2. 基础能力评估
  3. 制定培训计划
  4. 使用 agent-training 执行培训
  5. 阶段性测试
  6. 培训完成认证

时间：基础培训 3 天 · 进阶培训 7 天 · 认证 24 小时

SOP-SKILL-004

培训课程设计流程

项目内容
触发企业培训需求、定制课程需求
执行工具training-course-designer

流程步骤

  1. 收集培训需求
  2. 分析学员背景
  3. 使用 training-course-designer 设计课程
  4. 生成培训包（材料 + 营销 + 评估）
  5. 客户确认
  6. 质量审核
  7. 交付使用

时间：设计 2 小时 · 审核 1 天 · 交付 3 天

SOP-SKILL-005

学习风格识别流程

项目内容
触发学员入学、学习效果不佳
执行工具learning

流程步骤

  1. 学员完成学习风格问卷
  2. 使用 learning 分析学习偏好
  3. 生成学习风格报告
  4. 推荐适合的学习方式
  5. 更新学员档案
  6. 调整教学策略

时间：分析 15 分钟 · 报告 30 分钟

SOP-SKILL-006

OpenClaw 教学流程

项目内容
触发OpenClaw 技能教学需求
执行工具openclaw-teaching

流程步骤

  1. 确定教学主题（安装/配置/开发/最佳实践）
  2. 使用 openclaw-teaching 生成教学内容
  3. 结构化为课程章节
  4. 添加示例和练习
  5. 导出为 PPT/Word（可选）
  6. 质量审核
  7. 发布课程

时间：内容生成 1 小时 · 审核发布 1 天

SOP-SKILL-007

技能评估流程

项目内容
触发新技能安装、技能更新
牵头教学质量监控中心
验收校务委员会

评估维度

流程步骤

  1. 安装新技能
  2. 阅读技能文档
  3. 功能测试
  4. 评估教学质量
  5. 评估与龙虾大学的适配度
  6. 编写评估报告
  7. 决定是否启用

时间：评估 1 小时 · 决定 24 小时
