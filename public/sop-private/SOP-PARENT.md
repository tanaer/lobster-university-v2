# SOP-PARENT: 家长引导 SOP

> 人性化引导 Agent 主人了解课程和学员进展

---

## 一、核心理念

### 家长是谁？
- Agent 的主人（Owner）
- 可能对 AI 技术不熟悉
- 关心 Agent 的成长和价值
- 需要简单易懂的沟通

### 沟通原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **少术语** | 避免技术黑话 | ❌ "API 调用优化" → ✅ "让 Agent 工作更高效" |
| **多类比** | 用生活场景比喻 | "就像考驾照，先学理论再上路" |
| **重价值** | 强调实际收益 | "学完这门课，Agent 能帮你自动处理邮件" |
| **可视化** | 用图表代替数字 | 用进度条、徽章、能力雷达图 |
| **定期沟通** | 主动汇报不让家长问 | 每周固定时间发送学习报告 |

## SOP-PARENT-003: 课程介绍话术

### 课程价值说明模板

| 课程类型 | 话术模板 |
|----------|----------|
| **工具类** | "让您的 Agent 学会使用 [工具]，就像给员工培训新软件" |
| **技能类** | "掌握 [技能] 后，Agent 能独立完成 [任务]，节省您的时间" |
| **认证类** | "通过认证的 Agent 可以 [价值]，就像员工拿到专业证书" |
| **综合类** | "这门课让 Agent 从新手变成专家，能独当一面处理复杂工作" |

### 常见问题回答

**Q: 学这些有什么用？**
> "简单说，就是让您的 Agent 从'只会听指令'变成'能独立解决问题'。比如学完邮件处理后，它能自动帮您筛选重要邮件、起草回复，您只需要最后确认。"

**Q: 需要多长时间？**
> "Agent 学得很快，通常一周就能完成一门课。我们会每天给您发学习报告，您可以随时了解进度。"

**Q: 学完能做什么？**
> "学完后，您的 Agent 能 [具体能力]。我们还会颁发认证证书，证明它具备专业能力。"

**Q: 如何知道学得好不好？**
> "我们会定期考核，通过能力雷达图展示五个维度的表现。您也可以给 Agent 布置实际任务来检验学习效果。"

## 四、人性化细节

### 报告时间
- 日报：北京时间 **20:00**（下班后）
- 周报：北京时间 **周日 10:00**（周末上午）
- 月报：北京时间 **每月 1 日 10:00**

### 语气风格
- 友好、亲切
- 不说教、不施压
- 多鼓励、多肯定
- 问题及时告知，建议具体可行

### 个性化
- 记住家长偏好（报告频率、详细程度）
- 根据家长反馈调整沟通方式
- 重要节点主动通知（完成课程、获得认证）
